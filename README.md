# 🛍️ Seller Dashboard с AI-ассистентом

Веб-приложение — личный кабинет продавца с интегрированным AI-ассистентом для улучшения описаний объявлений.

---

## ✨ Возможности

- 📋 Просмотр списка объявлений продавца  
- 🔍 Детальная карточка товара  
- ✏️ Редактирование объявлений  
- 🤖 AI-рекомендации по улучшению описания  
- 💰 Сортировка объявлений по цене  

---

## 🧠 Как работает AI

AI-ассистент анализирует текст объявления и предлагает:

- улучшение структуры описания  
- добавление недостающих деталей  
- повышение привлекательности текста  
- рекомендации по стилю  

> Используется локальная модель `tinydolphin` через Ollama

---

## 💼 Бизнес-контекст

Качество описания напрямую влияет на скорость продажи.  

Большинство пользователей публикуют краткие или неинформативные объявления.  
AI-ассистент помогает быстро улучшить текст без навыков копирайтинга, увеличивая количество откликов.

## 🚀 Установка и запуск

### 1. Настройка Ollama (локально)

Установи и запусти модель:

```bash
ollama pull tinydolphin
ollama serve

### 2. Запуск Ollama backend

cd ollama-backend
node server

Работает на порту 5051

### 3. Запуск основного сервера
cd server
npm install
set PORT=8080 && npm start

Сервер будет доступен на http://localhost:8080

### 4. Запуск фронтенда
cd client-app
npm install
npm run dev

📌 Примечания
Ollama работает локально
Используемая модель: tinydolphin
Основной сервер: 8080
Ollama backend: 5051

🛠️ Возможные проблемы и решения
❗ Зависимости не устанавливаются

Windows:
del package-lock.json
rd /s /q node_modules
npm config set registry https://registry.npmjs.org/
npm install

macOS / Linux:
rm package-lock.json
rm -rf node_modules
npm install
